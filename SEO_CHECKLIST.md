## 🎯 Anahtar Kelime Optimizasyonu

### Kullanılması Gereken Kelimeler

#### Ana Anahtar Kelimeler
```
✅ Mutlaka kullanın:
- emlak
- gayrimenkul
- satılık
- kiralık
- [şehir adı] (ör: istanbul, ankara)
```

#### Uzun Kuyruk Anahtar Kelimeler
```
✅ İçeriklere serpiştirilir:
- [şehir] satılık [oda sayısı] daire
  Örn: "istanbul satılık 3+1 daire"
  
- [ilçe] kiralık [tip]
  Örn: "kadıköy kiralık villa"
  
- [özellik] [tip] [konum]
  Örn: "deniz manzaralı daire beşiktaş"
```

#### Yerel SEO Anahtar Kelimeleri
```
✅ Lokasyon odaklı:
- [mahalle/ilçe] + emlak
- [mahalle/ilçe] + gayrimenkul
- [mahalle/ilçe] + satılık ev
- [mahalle/ilçe] + kiralık daire

Örnek:
- "kadıköy emlak ofisi"
- "şişli gayrimenkul danışmanı"
- "beşiktaş satılık ev"
```

## ✨ Pro İpuçları

### En Yaygın SEO Hataları
```
❌ YAPMAYIN:
- Anahtar kelime dolgusu (keyword stuffing)
- Duplicate content
- Düşük kaliteli backlink satın alma
- Görselsiz içerik
- Yavaş yüklenen sayfalar
- Mobil uyumsuzluk
- Broken linkler
```

### Hızlı Kazanımlar
```
✅ YAPIN:
- Tüm görsellere alt text ekle (1 saat)
- Meta açıklamaları optimize et (2 saat)
- Site hızını artır (4 saat)
- İç linkleme düzenle (3 saat)
- Google My Business oluştur (30 dk)
- Sosyal medya profilleri tamamla (1 saat)
```
